Install the Extension

Use the extension to sign human-authored content and publish to The Signal index.

Open GitHub Repo Return to Index
1

Build the extension

Run npm run build:extension to generate apps/extension/dist.

2

Load it in Chrome

Go to chrome://extensions, enable Developer Mode, click Load unpacked.

3

Point to live API

Open the extension Options and set API Base to https://thesignal-rho.vercel.app.
